# 桌面书签

一个常驻桌面的轻量待办/想法/时间线小工具。

## 已有功能

- 中间悬浮的「书签」形态
- 点击展开/再次收起
- 待办：新增、勾选、删除、清除已完成、拖拽排序
- 想法：快速记录、带时间戳
- 时间线：把待办与想法按时间归档展示
- 本地 JSON 持久化
- 常驻顶层、无边框、透明窗口

## 运行方式

```bash
cd D:\Hanako\desktop-bookmark
npm start
```

## 打包

```bash
npm run build
```

## 文件结构

- `main.js`：Electron 主进程
- `preload.js`：安全 IPC 桥
- `store.js`：本地 JSON 存储
- `renderer/`：界面层
- `assets/`：图标

## 设计说明

- 折叠态做成书签样式，默认居中出现
- 展开态是一个深色轻量面板
- 时间线参考 Notion 的“按时间组织内容”的思路，但更偏个人速记
